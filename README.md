## Blinkit Sales Analysis

An end-to-end sales analytics project built using **Python, MS SQL Server, and Power BI** to analyze Blinkit's retail sales data and uncover actionable business insights.

##  Project Overview

This project demonstrates the complete data analytics workflow, including data cleaning, SQL querying, exploratory data analysis (EDA), and interactive dashboard development.

##  Tech Stack

- Python (Pandas, NumPy, Matplotlib)
- MS SQL Server
- Power BI
- Excel

## Dashboard Highlights

- Total Sales
- Average Sales
- Customer Ratings
- Sales by Outlet Type
- Sales by Outlet Size
- Sales by Location
- Item Category Performance
- Fat Content Analysis
- Outlet Establishment Trends

##  Key Business Insights

- Identified top-performing outlet types and locations.
- Analyzed product category sales distribution.
- Compared sales across outlet sizes.
- Evaluated customer ratings and purchasing trends.
- Built an interactive dashboard for business decision-making.

##  Repository Contents

- `Blinkitsales_analysis.ipynb` – Data cleaning and exploratory data analysis using Python
- `Blinkit Grocery Data SQL Queries.docx` – SQL queries used for analysis
- `Blinkit analysis dashboard.pbix` – Interactive Power BI dashboard
